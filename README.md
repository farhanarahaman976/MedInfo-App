# Medicine Information App

A simple medicine information application built with HTML, CSS, and JavaScript.

## Features

- Displays a list of medicines with categories, descriptions, uses, and dosage information
- Search medicines by name, description, or usage
- Filter by category

## Files

- `index.html` — application shell and layout
- `styles.css` — styling for the app
- `app.js` — search, filter, and render logic

## Run locally

1. Open `index.html` in a web browser.
2. Use the search field and category selector to explore the sample medicines.

## Extend the app

- Add more medicine data
- Support medicine details pages
- Add backend storage or API integration
